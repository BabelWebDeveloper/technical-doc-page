Wprowadzenie

JavaScript to wieloplatformowy, zorientowany obiektowo język skryptowy. Jest to mały i lekki język. Wewnątrz środowiska hosta (na przykład przeglądarki internetowej) JavaScript może być połączony z obiektami swojego środowiska, aby zapewnić nad nimi kontrolę programową. JavaScript zawiera standardową bibliotekę obiektów, takich jak Array, Date i Math, oraz podstawowy zestaw elementów języka, takich jak operatory, struktury sterujące i instrukcje. Core JavaScript może być rozszerzany do różnych celów poprzez uzupełnianie go dodatkowymi obiektami; na przykład: Client-side JavaScript rozszerza podstawowy język, dostarczając obiekty kontrolujące przeglądarkę i jej obiektowy model dokumentu (DOM). Na przykład rozszerzenia po stronie klienta umożliwiają aplikacji umieszczanie elementów w formularzu HTML i reagowanie na zdarzenia użytkownika, takie jak kliknięcia myszą, wprowadzanie formularza i nawigacja po stronie. Server-side JavaScript rozszerza język podstawowy, dostarczając obiekty związane z uruchamianiem JavaScript na serwerze. Na przykład rozszerzenia po stronie serwera pozwalają aplikacji komunikować się z bazą danych, zapewniać ciągłość informacji z jednego wywołania aplikacji do drugiego lub wykonywać operacje na plikach na serwerze.
